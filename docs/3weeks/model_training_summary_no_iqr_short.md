IQR 제거 없이 전체 데이터를 사용하여 학습한 결과를 요약한 보고서입니다.

# 📌 1. 전체 실행 개요
- IQR 제거 없이 원본 데이터 100% 사용
- 도시별 3개 모델(RandomForest / XGBoost / LightGBM) 학습
- 전체 실행 시간: **약 787.9초 (~13.1분)**

---

# 🇰🇷 2. Seoul (Feature Mode: ALL)

## 📊 데이터 개요
- 전체 행 수: **14,581,612**
- 사용 피처: 36개  
- Train / Val / Test = 10,207,130 / 2,187,241 / 2,187,241

## ⭐ 모델별 성능 요약

| 모델 | Test RMSE | Test R² |
|------|-----------|---------|
| **LightGBM** | **7.481** | **0.693** |
| XGBoost | 8.182 | 0.633 |
| RandomForest | 9.639 | 0.491 |

### ✔ 결론
- LightGBM이 가장 안정적이며 최고 성능  
- XGBoost가 2위  
- RandomForest는 대규모 데이터에서 상대적으로 성능이 낮음

# 🇺🇸 3. Washington DC (Feature Mode: ALL)

- 전체 행 수: **1,402,340**
- 사용 피처: 42개

## ⭐ 모델별 성능 요약

| 모델 | Test RMSE | Test R² |
|------|-----------|---------|

- DC는 피처 품질 및 노이즈 영향으로 전체적인 성능 차이가 작음
