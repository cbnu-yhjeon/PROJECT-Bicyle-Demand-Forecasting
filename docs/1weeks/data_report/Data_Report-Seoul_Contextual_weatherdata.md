### 1-1. 결측치 해석 및 전처리 전략 제안

- **결측 비율이 0%에 가까운 컬럼**: 신뢰도가 높으므로 그대로 사용 가능.
- **결측 비율이 중간 수준(예: 5~30%)인 컬럼**:
  - 시계열 보간(선형 보간, 전/후 관측값 보간) 또는
  - 모델링 시 `결측 여부 플래그(feature)`를 추가하여 사용을 고려.
- **결측 비율이 매우 높은 컬럼(예: 50% 이상)**:
  - 해당 컬럼이 모델 성능에 매우 중요하지 않다면 **제거**도 검토.
  - 도메인 상 필수라면, 다른 기상 변수와의 관계를 활용한 예측 대체 기법도 고려 가능.

## 4. 시계열 연속성과 관측 공백

- 각 연도별 관측 건수는 앞선 오버뷰 리포트에서 확인한 연도별 건수 표를 참고할 수 있다.
- 일반적으로 ASOS 자료는 **1시간 혹은 10분 간격**으로 관측되므로, 특정 기간에 건수가 비정상적으로 적으면 해당 기간에 **관측 공백 또는 수집 실패 구간**이 존재할 수 있다.
- 필요 시, `지점 + 일시` 기준으로 **이상하게 비어 있는 시간대**를 탐지하는 추가 EDA를 수행하는 것이 좋다.

## 5. 모델링 관점에서의 데이터 활용성 평가

1. **전반적인 품질**
   - 결측/중복의 정도와 수치형 값 범위를 기준으로 볼 때, 시계열 기상 데이터로서 **모델 학습에 활용 가능한 수준**으로 보인다.
2. **주의할 포인트**
   - 강수량·적설·일사량처럼 **본질적으로 0이 많은 희소(sparse) 변수**는 분포가 매우 치우쳐 있을 수 있으므로, 로그 변환·이진화(비/눈 여부) 등도 고려할 수 있다.
   - 결측이 집중된 특정 지점이나 특정 기간이 있다면, 해당 구간에 대한 별도 처리(제외 또는 대체)가 필요하다.
3. **자전거 수요 예측과의 연결**
   - 기온, 강수량, 풍속, 습도 등은 자전거 수요와 밀접한 상관관계를 가질 가능성이 높으므로, 이들 컬럼의 품질이 특히 중요하다.
   - 시간 해상도(시간별/일별)를 따릉이 수요 데이터와 맞춰서 리샘플링/집계를 수행하는 과정에서, 결측/이상값 처리 전략을 일관되게 적용해야 한다.
