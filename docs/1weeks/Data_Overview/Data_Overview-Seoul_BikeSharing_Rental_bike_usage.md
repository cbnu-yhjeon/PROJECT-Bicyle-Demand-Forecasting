
# 📊 자전거 이용정보(시간별) 데이터 오버뷰 (상세 버전)
### 데이터 기간: 2020-01-01 ~ 2025-06-31  
### 총 행(Row): 12,404  
### 총 컬럼(Column): 12  

---

# 1. 데이터 구조 요약

이 데이터는 **자전거 수요를 시간 단위로 기록한 시계열 이용 정보**이며  
하루 24시간, 여러 대여소, 다양한 이용자 속성을 기반으로 구성됨.

### ✔ 구성요소

| 구성요소 | 설명 |
|---------|------|
| 시간 요소 | 날짜(`rental_date`), 시(`rental_hour`) |
| 공간 요소 | 대여소 ID(`station_id`), 대여소명(`station_name`) |
| 사용자 요소 | 성별(`gender`), 연령대(`age_group`), 대여유형(`rental_type`) |
| 행동 데이터 | 이용건수(`count`), 이용시간(`usage_time_min`), 이동거리(`distance_m`) |
| 활동량 데이터 | 운동량(`calories`), 탄소 절감(`carbon_reduction`) |

---

# 2. 컬럼 그룹별 상세 설명

## 🕒 2.1 날짜/시간 컬럼

| 컬럼명 | 설명 |
|--------|------|
| `rental_date` | 분석의 기준이 되는 날짜 |
| `rental_hour` | 특정 날짜의 시간대(0~23) |

→ 시계열 분석·주말/평일 패턴 분석에 매우 중요  
→ 기상데이터와 결합 시 필수 Key

---

## 📍 2.2 공간정보 컬럼

| 컬럼명 | 설명 |
|--------|------|
| `station_id` | 고유 식별자로 JOIN Key 역할 |
| `station_name` | 사용자 친화적 라벨, 중복·오타 가능 |

→ POI 데이터, 자치구 데이터와 매핑 가능  
→ 지도 시각화(Folium)와 결합하여 공간 패턴 분석 가능

---

## 👤 2.3 사용자 속성 컬럼

| 컬럼명 | 설명 |
|--------|------|
| `gender` | 이용자 성별, 대문자 표준화 필요 |
| `age_group` | 연령대 구간 |

→ 성별/연령대에 따른 이용 패턴 분석 가능  
→ 세그먼트 예측 모델링에 중요

---

## 🚲 2.4 이용량·활동량 데이터

| 컬럼명 | 의미 |
|--------|-------|
| `count` | 특정 조건(시간×대여소×타입)의 이용건수 |
| `distance_m` | 이동 거리 |
| `usage_time_min` | 이용 시간 |
| `calories` | 칼로리 소모 |
| `carbon_reduction` | 탄소 저감 값 |

→ Regression/Time-series 모델의 타깃 후보  
→ 운동·환경 정책 분석에 활용 가능

---

# 3. 데이터 탐색 시 고려 포인트

### ✔ 3.1 시간 기반 패턴
- 출근/퇴근 시간대의 피크 존재 가능  
- 주말·공휴일 패턴 필수 분석  
- 계절 패턴(온도/날씨/월별 변화) 포함

### ✔ 3.2 대여소 기반 패턴
- 상업지역/관광지/주거지역별 수요 차이  
- POI와의 공간적 상관관계 분석 가능  
- 교통시설(지하철역) 인근 대여소 수요 급증 가능

### ✔ 3.3 사용자 기반 패턴
- 젊은층(20~30대) 비중 높을 가능성  
- 성별 이용 차이가 존재할 수 있음  
- 대여권 종류에 따른 행동 패턴 분석 가능
