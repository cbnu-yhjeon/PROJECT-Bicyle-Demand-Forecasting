# 📌 데이터 출처 및 라이선스 정리 문서

자전거 수요 예측 머신러닝 프로젝트에서는 **UCI 머신러닝 레포지토리(UCI
ML Repository)**와\
**서울 열린데이터 광장(Seoul Open Data Plaza)**에서 제공하는 자전거 공유
시스템 데이터를 활용한다.\
본 문서는 각 데이터별 **출처 / 라이선스 / 사용 조건**을 정리한 것이다.

------------------------------------------------------------------------

## 1. UCI Machine Learning Repository

### 1-1. Bike Sharing Dataset (Washington D.C.)

-   **출처**: UCI ML Repository\
    https://archive.ics.uci.edu/dataset/275/bike+sharing+dataset
-   **원본 제공자**: Capital Bikeshare (Washington D.C.)
-   **저자**: Fanaee-T & Gama (University of Porto)
-   **라이선스**:
    -   Public Domain 수준의 공개 데이터\
    -   UCI 자체는 특별한 제한이 없으며 연구·교육 목적 자유 사용 가능
-   **사용 범위**:
    -   학술 연구 / 과제 / 비상업적 목적 자유 사용
    -   상업적 이용도 대부분 가능(원본이 공공 데이터 기반)
-   **주의 사항**:
    -   보고서/논문에서 **UCI + 원저자 + Capital Bikeshare** 출처 표기
        권장

------------------------------------------------------------------------

### 1-2. Seoul Bike Sharing Demand Dataset (UCI)

-   **출처**: UCI ML Repository\
    https://archive.ics.uci.edu/dataset/560/seoul+bike+sharing+demand
-   **데이터 제공자**: 연구진(아시아개발은행)
-   **라이선스**:
    -   연구·교육 목적 자유 사용
    -   상업적 이용은 제공자 정책에 따라 제한 가능
-   **주의 사항**:
    -   실제 서울시 따릉이 원본 데이터와 일치하지 않음 → UCI용 연구
        정제본
    -   서울 열린데이터 광장 데이터와 **병합 시 변수 정의 확인 필요**

------------------------------------------------------------------------

## 2. 서울 열린데이터 광장 데이터

서울시 열린데이터는 기본적으로 **공공누리(KOGL) 라이선스** 기반으로
제공되며,\
대부분 **출처 표기 시 자유 이용 가능**하다.

※ 각 데이터 페이지의 "이용허락범위(공공누리 유형)"에서 확인 가능\
- **제1유형(출처표시)**: 자유 이용 가능\
- **제3유형(출처표시 + 변경금지)**: 가공은 불가
