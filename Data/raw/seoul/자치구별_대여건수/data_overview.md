# 🚲 서울시 따릉이 – **자치구별 대여 건수 데이터 스키마**

이 데이터는 특정 연도·월 기준으로 **서울시 각 자치구별 따릉이 대여 건수를 집계한 데이터**입니다.  
서울 25개 자치구 + 기타(미분류)로 구성되어 있으며, 월 단위 수요 분석에 주로 활용됩니다.

---

## 📂 컬럼 설명 (Schema)

| 컬럼명 | 데이터 타입 | 설명 |
|--------|--------------|------|
| **년** | int | 집계 기준 연도 (예: 2021) |
| **월** | int | 집계 기준 월 (1–12) |
| **계** | int | 전체 대여 건수 (서울시 전체 합계) |

## 📘 활용 포인트

- 📈 **지역별 수요 분석**  
  자치구별 따릉이 사용 패턴 파악  
  (강남구·송파구·마포구 등 수요 집중 지역 탐지)

- 🗺️ **지역 특성 기반 모델링**  
  거주 인구/직장 인구, 기상 데이터와 결합하여  
  자치구 단위 예측 모델 구축 가능

- 🕵️ **추세 분석(연·월 변화)**  
  코로나 영향 분석, 계절별 패턴 분석 등 가능

- 🔧 **머신러닝 Feature 생성**  
  - 자치구별 비율(강남구 대여량 / 전체 대여량)  
  - 자치구 그룹 클러스터링  
  - 상위 3개 자치구 특징 추출 등
