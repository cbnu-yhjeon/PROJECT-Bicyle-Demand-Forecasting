# 🚲 서울시 따릉이 – **시간별 대여소 거치수량 데이터 스키마**

아래는 제공된 데이터(예: `일시`, `대여소번호`, `대여소ID`, `거치수량`)에 기반하여  
서울시 따릉이 **시간별 대여소 상태 데이터(거치수량 현황)** 의 스키마를 정리한 내용입니다.

---

## 📌 데이터 개요

이 데이터는 특정 시점(예: 2021-01-01 8:00)에  
각 따릉이 **대여소별로 사용 가능한 거치대 수(거치수량)** 를 기록한 데이터입니다.

수요 예측 또는 공급 배치 분석(Stations with low availability / high availability)에 활용할 수 있습니다.

---

## 📂 컬럼 설명 (Schema)

| 컬럼명 | 데이터 타입 | 설명 |
|--------|--------------|------|
| **일시** | `datetime` | 데이터가 수집된 시각. `YYYY-MM-DD HH:00` 형식의 시간 단위 데이터 |
| **대여소번호** | `int` | 각 대여소를 식별하는 고유 번호 |
| **대여소ID** | `string` | 대여소를 나타내는 문자열 ID (예: `"ST-484"`, `"ST-3"`) |
| **거치수량** | `int` | 해당 시점 기준 대여소에서 사용 가능한 자전거 거치대 수 |

---

## 📊 예시 데이터

| 일시 | 대여소번호 | 대여소ID | 거치수량 |

---

## 📘 분석 활용 포인트

- **시계열 기반 대여소 가용량 분석**  
  - 특정 시간대/요일에 대여소가 비거나 가득 차는 패턴 찾기  

- **수요 예측 모델 피처로 사용 가능**  
  - “거치수량”은 공급 상태를 나타내므로  
    향후 대여량 예측 시 중요한 변수로 활용 가능

- **좌표 데이터(STA_LAT/STA_LONG)와 결합 가능**  
  - 공간 기반 분석(핫스팟 탐색), 클러스터링 가능

---

## 📌 추가로 생성 가능한 파생 컬럼 예시

| 파생 컬럼 | 설명 |
|-----------|------|
| `hour` | 시(hour) 추출 |
| `weekday` | 요일 |
| `is_workingday` | 평일/주말 여부 |
| `station_availability_ratio` | 전체 거치대 대비 이용 가능 비율 |
