# 🚲 서울시 따릉이 – **대여소별 이용정보(월별) API 스키마**

서울시 열린데이터광장에서 제공하는 **대여소별 월간 이용 현황** 데이터의  
출력값(Output) 스키마를 정리한 문서입니다.

---

## 📌 공통 출력 필드 (Common Fields)

| 구분 | 필드명 | 설명 |
|------|--------|------|
| 공통 | `list_total_count` | 총 데이터 건수 (정상 조회 시 출력) |
| 공통 | `RESULT.CODE` | 요청 결과 코드 (API 호출 성공/실패 코드) |
| 공통 | `RESULT.MESSAGE` | 요청 결과 메시지 |

---

## 📂 대여소별 월간 이용정보 필드 (Monthly Station Usage Fields)

| 번호 | 필드명 | 설명 |
|------|---------|-------|
| 1 | `STATMN` | 연월(YYYYMMDD 형식이나 YYYYMM 사용 가능) |
| 2 | `STATIONNAME` | 대여소 이름 |
| 3 | `STATIONGRPNAME` | 대여소 그룹명 |
| 4 | `RENTCNT` | 해당 월의 총 대여 건수 |

---

## 📊 데이터 구조 예시

| STATMN | STATIONNAME | STATIONGRPNAME | RENTCNT |
|--------|--------------|----------------|---------|
| 202001 | 신촌로터리_01 | 신촌권역 | 12500 |
| 202001 | 여의도공원_03 | 여의도권역 | 9800 |
| … | … | … | … |

---

## 📌 데이터 활용 포인트

- **대여소별 월간 수요 분석**  
  - 인기 대여소 파악  
  - 월별 성수기/비수기 분석 가능  

- **시계열 기반 수요 패턴 연구**  
  - 월 단위 추세(Trend) 모델 구축  
  - ARIMA / Prophet 기반 월간 수요 예측

- **대여소 그룹(행정동/권역) 단위 비교 분석**  
  - `STATIONGRPNAME`을 기반으로 권역별 특성 분석 가능  
  - 배치 계획/운영 전략 수립에 활용

---

## 📌 파생 지표 예시

| 파생 컬럼 | 설명 |
|-----------|------|
| `rent_growth_rate` | 월별 증감률 |
| `avg_rent_per_day` | 월간 평균 1일 대여 건수 |
| `peak_month_flag` | 해당 대여소의 최고 수요 월 여부 |
